# Security Policy

## Supported Versions

The following versions will get security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 0.4.x   | :white_check_mark: |
| 0.3.x   | :x:                |
| 0.2.x   | :x:                |
| 0.1.x   | :x:                |

## Reporting a Vulnerability

If you find a vulnerability, simply [open an issue](../../issues/new) with the details, use the label `security`.
